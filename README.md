# 📈 TQQQ Signal Dashboard

나스닥 데이터를 기반으로 TQQQ 진입 신호를 매일 자동으로 계산해주는 대시보드입니다.

## 🔗 사이트 바로가기
👉 **https://jungguck.github.io/fun_stock**

## 📊 주요 기능
- TQQQ 매수 / 관망 / 청산 신호 (매일 자동 업데이트)
- QQQ 이동평균 (MA50 / MA200) 분석
- RSI · VIX 지표 확인
- 미국 국채 금리 (2년물 · 10년물 · 30년물) 모니터링
- 장단기 금리 역전 여부 확인

## ⚙️ 기술 스택
- Python (yfinance, pandas)
- GitHub Actions (매일 자동 실행)
- GitHub Pages (무료 호스팅)
- Chart.js (차트 시각화)

## ⚠️ 면책 고지
이 사이트는 참고용 정보만 제공합니다.
